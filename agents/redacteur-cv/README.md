# Rédacteur-CV Agent 🤖✍️

Agent IA de génération de CV optimisés pour des offres d'emploi spécifiques.

## 🎯 Mission

Le **Rédacteur-CV** est l'agent le plus complexe et critique du système Talentious. Il combine :
- L'analyse d'une offre d'emploi (depuis **Analyseur-Offre**)
- Le profil complet d'un utilisateur (depuis le **Backend**)

Pour produire un **CV optimisé en JSON** parfaitement adapté à l'offre cible.

## 🚀 Technologies

- **FastAPI** : Framework web asynchrone
- **Vertex AI Gemini 2.5 Flash** : IA générative (europe-west9, Paris)
- **Pydantic v2** : Validation des données
- **Docker** : Containerisation
